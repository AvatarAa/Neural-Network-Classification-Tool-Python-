# Recruiter Project Summary

Built a Python neural-network classification experiment for the Iris dataset using
scikit-learn's `MLPClassifier`. The project compares multiple network structures
and learning rates, standardizes input features, encodes class labels, trains on a
train/test split, and reports classification accuracy with loss-curve plots.

Key strengths:

- Demonstrates practical machine-learning workflow: preprocessing, model training,
  experimentation, evaluation, and visualization.
- Compares no hidden layer, one hidden layer, two hidden layers, and multiple
  learning rates.
- Uses a clean repository structure with source code, dataset, screenshots, and
  setup instructions separated for readability.
- Preserves the original neural-network behavior while making the project easier
  for reviewers to understand and run.
